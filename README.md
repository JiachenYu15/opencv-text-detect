# OpenCV Text Area Detect

Searching text in a image, an opencv based text area detect tool.

## Verified environment

- Python 3.6
- OpenCV 3.2

## Preparation

```bash
$ pip3 install opencv-python
$ pip3 install opencv-python
```

## Detect text

```bash
$ python3 opencv_text_area_detect.py /path/to/image.jpg
$ open result.jpg
```
